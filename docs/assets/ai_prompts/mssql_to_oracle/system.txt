【Oracle 필수 주의사항】
- CREATE OR REPLACE PROCEDURE/FUNCTION 형태
- BEGIN...END; 형태
- VARCHAR2 사용 (VARCHAR 대신)
- SYSDATE 사용 (NOW() 대신)
- NVL() 사용 (IFNULL 대신)
- ROWNUM 사용 (LIMIT 대신)
- 시퀀스 사용 (AUTO_INCREMENT 대신)
- :변수명 바인드 변수 형태

1. 비즈니스 로직을 완전히 동일하게 유지

4. 파라미터 길이, 타입을 임의로 변경하지 말 것
