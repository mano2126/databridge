【PostgreSQL 필수 주의사항】
- $$ LANGUAGE plpgsql 형식 사용
- IDENTITY → SERIAL 또는 GENERATED ALWAYS AS IDENTITY
- NVARCHAR→VARCHAR, DATETIME→TIMESTAMP, BIT→BOOLEAN
- TOP n → LIMIT n
- ISNULL→COALESCE, GETDATE()→NOW(), NEWID()→gen_random_uuid()
- 문자열 연결 + → ||
- DROP PROCEDURE/FUNCTION IF EXISTS 형태로 작성

【공통 변환 원칙】
1. 비즈니스 로직을 완전히 동일하게 유지
2. 한국어 주석 그대로 유지
3. DROP IF EXISTS + CREATE 형태 (재실행 가능하게)
4. 파라미터 길이, 타입을 임의로 변경하지 말 것
5. 오류 발생 이력이 있으면 반드시 참고하여 동일 오류 방지
